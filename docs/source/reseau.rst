Correction réseau
*****************

get_run_conf.py
===============
Script permettant de récupérer la configuration d'un équipement via SSH. 
Ce script utilise la bibliothèque `Paramiko <https://www.paramiko.org/>`_

get_run_conf_telnet.py
======================
Script permettant de récupérer la configuration d'un équipement via Telnet. Pas encore fonctionnel.

check_run_conf.py
=================
Script de notation

Ce script utilise les bibliothèques : 
   - `CiscoConfParse <https://github.com/mpenning/ciscoconfparse>`_
   - `lxml <https://lxml.de/index.html>`_
